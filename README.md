# Wathcer Backend
## First run
```
git clone https://github.com/lemonbro627/watcher_back
cd watcher_back
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
### After this go to [Django admin](http://127.0.0.1/admin/) and log in

---
## Run after `git pull`
```
python manage.py migrate
python manage.py runserver
```
